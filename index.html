HOW IT WORKS

One gateway. Every AI request. Full control.

TraceLayer is an OpenAI-compatible reverse proxy. Integration is a one-line URL change. Governance is immediate.

1

Authenticate & Identify

Every request is validated via OIDC/JWT. TraceLayer resolves which application and which employee is making the request — from day one.

2

Evaluate Policy

The OPA policy engine checks the request against your compliance rules in under 50ms. Only approved models, providers, and use cases are permitted.

3

Redact Sensitive Data

Before a single token leaves your perimeter, Presidio scans the prompt for PII, PCI, and financial identifiers — and redacts them. In-perimeter, no third-party data transfer.

4

Route & Enforce Budgets

The approved provider receives the clean, redacted prompt. Real-time budget checks prevent overspend before requests are sent.

5

Write Immutable Audit Record

Every request, policy decision, and redaction action is logged to tamper-evident WORM storage (AWS S3 Object Lock). Cryptographic Merkle roots verify completeness daily.

PLATFORM FEATURES

Enterprise-grade governance. Developer-first integration.

Policy-as-Code Engine

Powered by Open Policy Agent (OPA). Policies are versioned, approved, and deployed as code — with full change history and instant rollback. UK Finance baseline policy pack included.

In-Perimeter DLP

Presidio runs entirely within your network boundary. PII, PCI, and financial identifiers are detected and redacted before they ever leave your environment. Safe Streaming mode handles real-time responses.

Immutable Audit Trail

WORM-compliant storage via AWS S3 Object Lock. Daily Merkle root cryptographic chaining provides tamper-evident proof of completeness. Export signed evidence reports for auditors in PDF or CSV.

GDPR-Native Pseudonymisation

Audit logs store a non-identifiable subject_id UUID — never raw PII. Erasure requests are fulfilled via crypto-shredding, without deleting a single audit record.

Real-Time Cost Governance

Atomic reservation-based budget enforcement. Hard caps per cost centre prevent runaway spend. Configurable alert thresholds notify Finance before limits are breached. Per-provider cost dashboards.

Drop-In API Compatibility

TraceLayer is 100% OpenAI API-compatible. Integration requires changing one line of code — your base URL. Existing OpenAI SDKs work without modification, including streaming.

Multi-Tenant Architecture

Full data and policy isolation between tenants enforced at the database level via Row-Level Security. Each tenant has their own KMS encryption key. No shared state in the data path.

SIEM Integration

Forward audit events to Splunk or Microsoft Sentinel in real time via HTTP Event Collector webhooks. Your SOC gets full visibility into AI activity without any new tooling.

UK Data Residency

Deployed by default in AWS eu-west-2 (London). All data processing, audit storage, and encryption key management stays within UK borders. Purpose-built for FCA-regulated firms.
